02. Estructuras de Control

Aprende a controlar el flujo de tu aplicación usando condicionales y bucles.

¿Qué vamos a aprender?

1. Condicionales (if/else): Ejecutar código solo si se cumple una condición.

2. Switch: Una forma más limpia de hacer múltiples "if" cuando evaluamos una sola variable.

3. Bucles (while/for): Repetir una acción varias veces de forma automática.

// Juego de adivinar el número (Revisa la consola F12)
if (numeroUsuario === numeroMaquina) {
  console.log("¡Ganaste!");
} else {
  console.log("Sigue intentando...");
}
