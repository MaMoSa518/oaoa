05. Manipulación del DOM

DOM = Document Object Model. Aprende a crear, modificar y eliminar elementos HTML usando solo JavaScript.

Ejemplo Práctico

El recuadro de abajo es un <div id="contenedor-dom"> vacío en el HTML, pero lo llenaremos y modificaremos dinámicamente usando JavaScript.

Este es el párrafo original escrito en HTML.

¿Qué estamos haciendo con JS?

  • Seleccionamos el párrafo usando document.getElementById() o document.querySelector().
  • Modificamos su texto interno con .textContent o .innerHTML.
  • Creamos nuevos elementos desde cero (títulos, imágenes) usando document.createElement().
  • Agregamos estilos dinámicos a través de la propiedad .style.
