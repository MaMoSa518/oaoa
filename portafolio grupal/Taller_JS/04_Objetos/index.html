04. Objetos

Aprende a modelar información compleja usando objetos literales, métodos y desestructuración.

¿Qué vamos a aprender?

1. Objetos Literales: Cómo almacenar datos en pares de "clave-valor".

2. Métodos en Objetos: Funciones que pertenecen a un objeto y describen su comportamiento.

3. Desestructuración (ES6+): Una forma moderna y limpia de extraer variables desde el interior de un objeto.

// Objeto Literal
const alumno = {
  nombre: "Ana",
  edad: 18,
  curso: "Desarrollo Web"
};

// Desestructuración
const { nombre, curso } = alumno;
console.log(nombre); // Imprime: "Ana"

Abre el archivo mainObjetos.js y la consola (F12) para ver cómo funciona en la práctica.
