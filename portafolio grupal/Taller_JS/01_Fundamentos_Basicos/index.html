01. Fundamentos Básicos

Variables, tipos de datos, operadores matemáticos y la consola del navegador.

Instrucciones

1. Abre las Herramientas de Desarrollador de tu navegador (Presiona F12 o haz clic derecho > Inspeccionar > Consola).

2. En la consola verás los resultados de los scripts asociados a este módulo (inicio.js y mainBas.js).

3. Abre esos archivos en tu editor de código para leer las explicaciones de cómo funcionan las variables `let` y `const`, los tipos de datos y los operadores básicos.

// Ejemplo de código ES6 que encontrarás en los archivos:
const saludo = "¡Hola Mundo!";
console.log(saludo);
