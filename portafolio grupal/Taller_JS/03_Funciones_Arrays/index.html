03. Funciones y Arrays

Domina la reutilización de código y las colecciones de datos modernas.

¿Qué vamos a aprender?

1. Funciones Modernas: Diferencias entre funciones tradicionales y Arrow Functions (Funciones flecha).

2. Arrays (Arreglos): Creación e iteración sobre listas de datos.

3. Métodos de Arrays (ES6+): El poder de map, filter y reduce para transformar datos sin usar ciclos manuales.

// Función Flecha (Arrow Function)
const multiplicar = (a, b) => a * b;

// Método Map (Transformar un Array)
const numeros = [1, 2, 3];
const dobles = numeros.map(num => num * 2); // [2, 4, 6]

Abre el archivo mainFunciones.js y la consola (F12) para ver los ejemplos paso a paso.
